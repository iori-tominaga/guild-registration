# 要件定義書 — 冒険者ギルド登録アプリ

**バージョン**: 1.4.0  
**最終更新**: 2026-05-01  
**ステータス**: リリース済み（GitHub Pages）

---

## 1. プロジェクト概要

### 1.1 目的
冒険者ギルドへのユーザー登録体験を、ドラゴンクエスト風のRPGインターフェースで提供する。登録後はギルド本部ハブから各機能（装備・図鑑・呪文など）へ遷移できる。

### 1.2 背景
「コードクエスト」学習課題の達成物として作成。GitHub Pages で公開。

### 1.3 対象ユーザー
- PC・スマートフォンを利用するすべてのユーザー
- RPG・ゲームに親しみのあるユーザーを主なターゲットとする

### 1.4 公開URL
`https://iori-tominaga.github.io/guild-registration/`

## 3. 機能要件

### 3.1 登録フロー（index.html）

#### Step 1: 名前入力
- 冒険者名をテキスト入力できる（最大10文字）
- 入力文字数をリアルタイムに表示する（例: `3/10`）
- 入力が空の場合、「けってい」ボタンは無効化される
- Enterキーでも次のステップへ進める

#### Step 2: 職業選択
- 9種の職業から選択できる（せんし / まほうつかい / そうりょ / とうぞく / しょうにん / ぶとうか / うたうたい / けんじゃ / その他）
- ↑↓キーによるカーソル移動、Enterで決定、Escで戻る
- カーソル位置の職業のステータス（力・素早・守備・魔力・回復）をプレビュー表示する

#### Step 3: 自己紹介入力
- 自己紹介文をテキストエリアに入力できる（最大200文字）
- 「スキップ」ボタンで入力なしに次へ進める

#### Step 4: 確認画面
- 入力した名前・職業・自己紹介をカード形式でプレビュー表示する
- 「はい」で登録を確定、「いいえ」で前へ戻る

#### Step 5: 登録完了
- 登録完了メッセージをタイプライター演出で表示する
- 冒険者ギルドカードを発行・表示する（名前・職業・ランク・Lv・自己紹介・ステータス・登録日・ID）
- パーティクルエフェクトとホワイトフラッシュ演出を再生する
- キャラクターデータを localStorage の `dq-character` キーに保存する
- 登録履歴を localStorage の `dq-guild-history` キーに追記する（最大50件）
- 「ギルド本部へ」ボタンで board.html へ遷移する

#### 登録履歴閲覧
- 過去の登録カードを一覧表示できる（最大50件）
- 履歴がある場合、タイトル下に「📜 きろく（N件）」ボタンを表示する
- 確認ダイアログを経て履歴を全削除できる

### 3.2 ギルド本部（board.html）

#### キャラクター情報パネル
- localStorage の `dq-character` からキャラクターデータを読み込み表示する
- 表示項目：名前・職業アイコン・ランク・Lv・HP/MP/ATK/DEF/SPD
- ステータスはバーグラフで可視化する
- 装備ボーナス（`dq-equipment`）を加算して表示し、上昇分を緑の▲で示す
- キャラクターが未登録の場合、登録ページへのリンクを表示する

#### 個人クエストログ
- タスクをテキスト入力で追加できる（最大50文字）
- Enterキーまたは「＋」ボタンで追加する
- チェックボックスで完了/未完了を切り替えられる
- 「✕」ボタンでタスクを削除できる
- データは localStorage の `dq-quest-log` に永続化される

#### 冒険の場所
- 各クエストで追加される機能へのリンクカードを並べる
- 未解放のリンクは🔒ロック表示とする
- Quest 03 完了後、「そうびの　ま」（equipment.html）が解放される

### 3.3 装備の間（equipment.html）

#### 装備データ
武器5種・盾3種・鎧3種・アクセサリ3種（計14種）を所持する。各装備は以下を持つ：

| 属性 | 内容 |
|------|------|
| id | 識別子（文字列） |
| name | 装備名（ひらがな） |
| slot | weapon / shield / armor / accessory |
| icon | 絵文字 |
| desc | 説明文 |
| bonus | HP/MP/ATK/DEF/SPD の補正値 |

#### スロットパネル
- 4スロット（武器・盾・鎧・アクセサリ）の現在の装着状況を一覧表示する

#### ステータスプレビュー
- 装備ボーナスを加算したステータスをリアルタイムで表示する
- 装備による上昇分を緑の▲で明示する

#### 装備一覧
- スロット別にグルーピングされた全装備を表示する
- 「そうび」ボタンで装着、「はずす」ボタンで解除する
- 装着済みの装備はハイライト表示する
- 装着時にゴールドフラッシュ演出と上昇音を再生する
- 装備状態は localStorage の `dq-equipment` に保存される

### 3.4 UI・演出共通要件

| 要件 | 内容 |
|------|------|
| フォント | Press Start 2P（ドット文字フォント） |
| ウィンドウ枠 | ドラゴンクエスト風（濃紺背景・二重ボーダー） |
| 背景 | 星空（ラジアルグラデーション）+ 暗色背景 |
| タイプライター演出 | メッセージが1文字ずつ表示される |
| サウンド | Web Audio API による8bitサウンド |

### 3.5 レスポンシブ要件

| ブレークポイント | 対応内容 |
|-----------------|----------|
| 〜520px（スマホ） | input/textarea の font-size を16px（iOS自動ズーム防止） |
| 〜520px（スマホ） | ボタンの最小高さを44px（タッチターゲット確保） |
| 〜520px（スマホ） | 職業選択のステータスパネルを非表示 |
| ノッチ付き端末 | safe-area-inset-top による上部余白確保 |

## 4. localStorage スキーマ

| キー | 型 | 内容 |
|------|----|------|
| `dq-character` | Object | 最新の登録キャラクター情報 |
| `dq-guild-history` | Array | 登録履歴（最大50件、新しい順） |
| `dq-quest-log` | Array | 個人クエストログ |
| `dq-equipment` | Object | 装着中の装備ID（スロットをキーとする） |
| `dq-bestiary` | Object | `{ defeatedMonsterIds: [] }`（Quest 06 バトルで更新） |

---

## 5. 非機能要件

| 区分 | 要件 |
|------|------|
| パフォーマンス | 初回表示3秒以内（Google Fonts・CDN読み込み含む） |
| ブラウザ対応 | Chrome / Safari / Firefox / Edge の最新版 |
| モバイル対応 | iOS Safari / Android Chrome で正常動作すること |
| セキュリティ | 入力値はクライアント内のみで処理し、外部送信しない |

---

## 6. 制約・除外事項

- バックエンド・データベースは持たない
- ユーザー認証は行わない
- 多言語対応は対象外（日本語のみ）
