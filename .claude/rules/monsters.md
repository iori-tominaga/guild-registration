# モンスター設計ルール

## 種別定義

| 種別 | 特徴 |
|------|------|
| スライム系 | HP高め・ATK/DEF低い・初心者向け |
| 魔獣系 | ATK高め・SPD高い・バランス型 |
| 不死系 | DEF高め・HP高い・魔法に弱い |
| ドラゴン系 | 全体的に高ステータス・上級者向け |
| 人型 | バランス型・特殊能力あり |
